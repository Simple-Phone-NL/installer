## 💡 Quick Reference

### To Add a New Button
1. In `setup_ui()`, add:
   ```python
   self.btn_my_action = QPushButton("My Button")
   self.btn_my_action.clicked.connect(self.on_my_action)
   right_layout.addWidget(self.btn_my_action)
   ```

2. Add handler:
   ```python
   def on_my_action(self):
       for serial in self.selected_devices:
           # Your logic here
           pass
   ```

### To Run a Command on Device
```python
success, output = self.device_manager.run_command(
    f"{self.device_manager.adb} shell your_command",
    serial
)
if success:
    self.log(f"✓ Success: {output}")
else:
    self.log(f"✗ Error: {output}")
```

### To Get Device Info
```python
state = self.device_manager.get_device_state(serial)
name = self.device_manager.get_device_name(serial)
self.log(f"Device {serial}: {name} (State: {state})")
```

## 🔧 Configuration

Most settings are in `main_gui.py`:

| Setting | Location | Default | Tip |
|---------|----------|---------|-----|
| Device scan interval | `DeviceDetectorThread.run()` | 2 sec | Lower = faster, higher = less CPU |
| Command timeout | `DeviceManager.run_command()` | 30 sec | Increase for slow connections |
| Download directory | `DeviceManager.__init__()` | "downloads" | Can change to any path |
| ADB binary path | `DeviceManager.__init__()` | "apt/adb" | Must exist in apt/ folder |

## ❓ FAQ

**Q: Why isn't my device showing up?**  
A: Ensure USB debugging is enabled. Click "Refresh Devices" button.

**Q: Can I add my own buttons?**  
A: Yes! Copy the pattern from existing buttons in `setup_ui()` and add handlers.

**Q: How do I implement custom ROM installation?**  
A: Edit `on_manual_install()` method. See CUSTOMIZATION.md for examples.

**Q: What if a device gets stuck?**  
A: Click "Reboot to System" to force it back to normal mode.

**Q: Can I run this on Mac/Linux?**  
A: Yes! PyQt6 and adb/fastboot work on all platforms. Just ensure adb/fastboot binaries are available.
