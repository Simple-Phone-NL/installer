## Installing Python (if needed)

If you don't have Python installed:

1. Go to https://www.python.org/downloads/
2. Download the latest version for your computer
3. Run the installer and follow the instructions
4. Make sure to check "Add Python to PATH" during installation

## Download the Installer

1. Go to https://github.com/Simple-Phone-NL/installer/releases
2. Download the latest ZIP file (it will be named something like `installer.zip`)
3. Extract the ZIP file to a folder on your computer 

## How to Use

1. **Connect the device** 

2. **Run the installer.py file**:

3. **Carefully follow the on-screen instructions**:
   - The installer will ask you to choose an option. The first time running this software you should make sure to select the option to update the software.
   - To select an option just enter a number and then press enter, or only press enter.
   - It will check if the device is connected
   - Follow the prompts carefully - it will tell you when to press buttons on the device

4. **Important steps on your device**:
   - When prompted, you will still need to perform certain actions on the device that can not yet be automated.

5. **Finish**: Once done, select "Reboot System Now" on the device
